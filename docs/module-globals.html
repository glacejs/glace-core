<abstract, inner> baseTest(name [, opts] [, fixtures], func)

Basis for any test case.

If test with the same was registered already, this test will be omitted with corresponding error in log.

If retries amount is specified and this test was failed, the test will be added to queue in separated session with title containing Retry and its number.
